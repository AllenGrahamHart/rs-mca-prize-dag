# e22_staircase_injectivity

- **status:** CONDITIONAL
- **closure:** proof

## Statement

The E22 quotient-coset staircase parametrization is injective up to the
declared quotient equivalences, and its image has no hidden overlap with the
planted sunflower codewords.

## Falsifier

Two inequivalent staircase parameter sets yielding the same degree-`<k`
codeword, or a non-degenerate staircase parameter set yielding a planted
codeword.
