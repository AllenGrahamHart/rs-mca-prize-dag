# sov_obstruction_equidistribution

- **status:** CONDITIONAL
- **closure:** proof

## Statement

For official-shape rows and `h in (20,A]`, the first obstruction map
`O_{h-1}` on anchored cores has sufficiently small fibers. Equivalently, the
value set of `O_{h-1}` is large enough to prevent a super-budget concentration
of active cores in any one value fiber.

## Falsifier

A calibration or official-shape row where `O_{h-1}` has a fiber above the
active-core budget, or a structural family of anchored cores on which
`O_{h-1}` is constant.
