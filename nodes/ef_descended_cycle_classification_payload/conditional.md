# conditional: ef_descended_cycle_classification_payload

## Predicate nodes

- `ef_descended_cycle_inventory_soundness`
- `ef_descended_cycle_inventory_payload`

## Claim

Conditional on the actual descended-cycle inventory, every `B`-defined
pole-free horizontal cycle after full-orbit descent is classified as
base-descended, proper-subfield/tower-confined, or noncontainment-degenerate.

## Proof

The predicate `ef_descended_cycle_inventory_payload` supplies an inventory
covering every descended `B`-defined pole-free horizontal cycle. Each entry
has a verified certificate of one accepted label: base-descended,
proper-subfield/tower-confined, or noncontainment-degenerate.

The proved predicate `ef_descended_cycle_inventory_soundness` says that such a
complete verified inventory implies exactly this classification payload.
Therefore this node follows.
