# dihedral_staircase

- **status:** PROVED
- **closure:** proof

## Statement

For a fixed received pair `(u,v)`, assume `k + 3j <= n`. Then the number of
aligned exact `j`-supports over all slopes is at most `n + 1`. In particular,
the number of aligned inversion-closed, hence dihedral, exact `j`-supports is
at most `n + 1`.

This is the scoped deep-regime staircase lemma. It does not assert the old
clean-corridor-wide dihedral budget statement; that role was folded into
`R_post` by the poly-forcing compiler.
