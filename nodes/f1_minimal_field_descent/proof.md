# proof: f1_minimal_field_descent

- **status:** PROVED
- **closure:** proof

## Statement (this node)

Let `B = F_p(D)` be the field generated by the domain data and `F = F_{p^N}` the
ambient sampling field, `B <= F`. For a bad slope produced by a pair `(u,v)`,
its **pencil data** `x` (the syndrome-built pencil coefficients `a(l), b(l)`
and the aligned slope `Z = -a_r/b_r`, taken up to the pencil's projective
scaling ambiguity) has a **well-defined minimal field of definition** `K` with
`B <= K <= F`; and the case split

```
(I)  K = B            (II)  B < K < F            (III)  K = F
```

is exhaustive by field theory. This is the field-descent bookkeeping that feeds
the S6 trichotomy (base-rational / subfield-confined / genuinely-`F`-valued).

## 0. Why this is not automatic — and why it nonetheless holds

The falsifier to defeat is *composite ambiguity*: a datum defined over two
intermediate fields `K_1, K_2` with **no** common sub-field-of-definition, so
that no single minimal `K` exists. For a general base scheme this can happen
(a point can be defined over `K_1` and over `K_2` but not over `K_1 ∩ K_2` when
descent fails). The content of this node is that for our situation it **cannot**,
for one structural reason: `F/B` is a finite-field extension, hence **Galois with
cyclic Galois group**, and cyclic (indeed abelian) descent always produces a
unique minimal field of definition. We prove exactly this.

## 1. The subfield lattice of a finite field is the divisor lattice [banked, restated]

`F = F_{p^N}`. Its subfields are exactly `{ F_{p^d} : d | N }`, and

```
d |-> F_{p^d}   is an order-isomorphism   (Div(N), |)  ->  (Subfields(F), ⊆),
F_{p^{d_1}} ∩ F_{p^{d_2}} = F_{p^{gcd(d_1,d_2)}},
F_{p^{d_1}} · F_{p^{d_2}} = F_{p^{lcm(d_1,d_2)}}.                          (L)
```

In particular the poset of subfields is a **lattice** closed under intersection.
`Gal(F/F_p) = <Frob>`, `Frob(x)=x^p`, is **cyclic of order `N`**; via the Galois
correspondence, `Gal(F/F_{p^d})` is the unique subgroup of order `N/d`, and every
subgroup of a cyclic group is normal and uniquely determined by its order. Write
`B = F_{p^b}`, `b | N`. (`B` is a field because it is generated by a subset of
`F`; `b = [B:F_p]`.)

## 2. Field of definition of a datum, and its minimality [complete]

Let `x` be the pencil datum, presented as a projective point
`[a_0 : ... : a_m]` over `F` (the pencil coefficients up to common scaling),
together with the derived slope `Z = -a_r/b_r in F`. Define its **stabilizer**

```
Stab(x) = { sigma in Gal(F/F_p) : sigma fixes x up to the pencil's scaling
            ambiguity },  equivalently  { sigma : sigma(a_i/a_j) = a_i/a_j  for all i,j }.
```

`Stab(x)` is a subgroup of `Gal(F/F_p)` (it is the stabilizer of a point under a
group action: closed under composition and inverse). Because `Gal(F/F_p)` is
cyclic, `Stab(x)` is the unique subgroup of its order, say of index `d_x | N`,
and we set

```
K_0(x) := F^{Stab(x)} = F_{p^{d_x}}     (the minimal field of definition over F_p).
```

**Descent (well-definedness).** `K_0(x)` is a field of definition of `x`: the
ratios `a_i/a_j` are fixed by `Stab(x)`, hence lie in `F^{Stab(x)} = K_0(x)`, so
`x` is representable with coordinates in `K_0(x)` (clear denominators by any
nonzero `a_j`). Conversely, if `x` is defined over some subfield `K' = F_{p^{d'}}`
then every `sigma in Gal(F/K')` fixes the `K'`-rational ratios, so
`Gal(F/K') <= Stab(x)`, i.e. (Galois correspondence, orders) `d' ` is a multiple
of... more precisely `Gal(F/K') <= Stab(x)` gives `K_0(x) = F^{Stab(x)} <= F^{Gal(F/K')} = K'`.
Thus **every** field of definition contains `K_0(x)`, so `K_0(x)` is the unique
minimal one. In lattice terms: the set `{ K' : x defined over K' }` corresponds
to `{ subgroups <= Stab(x) }`, which has a unique maximum `Stab(x)`, hence the
field set has a unique minimum `K_0(x)` — **no composite ambiguity is possible**,
because a chain-condition lattice arising from a cyclic group has unique
extremal elements. This is where cyclicity of `Gal(F/F_p)` is used; for a
non-abelian Galois group the intersection of two fields of definition need not be
one, and the falsifier would be live.

## 3. Relativising to the base `B`, and the trichotomy [complete]

We want the minimal field of definition **containing `B`** (the pair's slopes
are always at least `B`-rational data-wise; `B` is the ledger's floor). Set

```
K := B · K_0(x) = F_{p^{lcm(b, d_x)}}      (by (L)).                       (K)
```

By (L) this is the unique smallest subfield of `F` containing both `B` and
`K_0(x)`; equivalently `K = F^{Stab(x) ∩ Gal(F/B)}`, the fixed field of the
stabilizer intersected with `Gal(F/B)`. It is well-defined and satisfies
`B <= K <= F` by construction: `B <= K` since `b | lcm(b,d_x)`, and `K <= F`
trivially.

**Exhaustiveness of the case split.** `K` is a single intermediate field of the
interval `[B, F]` in the subfield lattice. Any element of a poset interval
`[B,F]` is either equal to the bottom `B`, equal to the top `F`, or strictly
between; these three cases are mutually exclusive and cover everything. Hence

```
K = B          <=>  lcm(b, d_x) = b   <=>  d_x | b   (datum B-rational)      : case (I)
B < K < F      <=>  b | lcm(b,d_x) | N properly on both ends                 : case (II)
K = F          <=>  lcm(b, d_x) = N   (genuinely F-valued)                   : case (III)
```

is an exhaustive, mutually exclusive trichotomy — this is the S6 case split
(I = base-rational, paid by the base ledgers; II = subfield-confined to some
`B < K < F`, handled by the S6 tower recursion one level up; III = genuinely
`F`-valued, requiring the extension-pole / `F`-valued-word mechanism). ∎

## 4. Consistency with S6 §1 (the "free lift")

S6 §1's elementary observation — *a `B`-rational pair produces only `B`-rational
slopes* — is exactly case (I) of this trichotomy: if the pair (hence its
syndromes, hence all `a(l), b(l)`) is `B`-rational then `Stab(x) >= Gal(F/B)`,
so `d_x | b`, so `K = B`. The trichotomy generalises this to the exact statement
of *which* intermediate field the datum descends to, which is all S6's
classification needs.

trichotomy case. All checks PASS.
