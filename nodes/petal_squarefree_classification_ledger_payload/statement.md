# petal_squarefree_classification_ledger_payload

- **status:** TARGET
- **closure:** proof

## Statement

Construct the actual Petal squarefree classification ledger covering every
squarefree realizable locator point in the residue-line kernels by:

- charged class records with citations to already paid families; or
- uncharged class records with polynomial bounds whose exponents are
  independent of the excess `c`.

The number of uncharged class records must be bounded independently of `c`.

## Falsifier

A squarefree realizable kernel family missing from the ledger, charged without
a paid citation, or uncharged without a `c`-independent polynomial bound.
