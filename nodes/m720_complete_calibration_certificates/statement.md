# m720_complete_calibration_certificates

- **status:** PROVED
- **closure:** proof

## Statement

Complete calibration cells for `h=7..20` have zero unpaid non-toral active
cores. Any n=1024 window-slice result is explicitly labeled as slice-local and
is not used as a global zero certificate.

## Falsifier

A complete calibration cell containing an unpaid non-toral active core, or a
certificate that treats an incomplete window slice as a complete cell.
