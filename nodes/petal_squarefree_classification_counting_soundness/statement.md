# petal_squarefree_classification_counting_soundness

- **status:** PROVED
- **closure:** proof

## Statement

Assume that, for every coset-chart corridor cell, the squarefree realizable
locator points inside the residue-line kernel are partitioned into:

- charged classes already paid elsewhere; and
- finitely many uncharged classes, each bounded by `C_i n^{A_i}` where the
  exponent `A_i` is independent of the excess `c`.

If the number of uncharged classes is itself bounded independently of `c`,
then the total uncharged squarefree realizable kernel points are bounded by a
polynomial whose exponent is independent of `c`.

## Falsifier

A finite classification with every uncharged class uniformly polynomial in
the corridor parameters but whose union has exponent depending on `c`.
