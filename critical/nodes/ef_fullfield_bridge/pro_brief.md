# Pro follow-up brief — the ef_fullfield_bridge (the S9 typecheck)

*Continuation of your pole-free cycle exclusion proof. You closed the Galois
descent unconditionally and isolated ONE seam. This brief is that seam, with
the s6 definitions pinned so it can be resolved or shown to be a genuine new
ledger class.*

## Recap
You proved: a B-defined pole-free horizontal cycle decomposes by Galois
stabilizer into base-descended / tower-confined / (trivial-stabilizer). In the
trivial-stabilizer noncontainment case you invoked ext_pole_floor to force the
component onto the pole divisor D, contradicting pole-freeness. The only
unverified step was the SCOPE MATCH: that such a component is an "extension
fiber to which ext_pole_floor applies." That is this node, ef_fullfield_bridge.

## The s6 machinery (pinned)
- B = F_p(D) the field generated by the evaluation domain D; F >= B the ambient
  sampling field; a "row" is intermediate B <= E <= F.
- The alignment pencil is LINEAR with coefficients a(l), b(l) built from the
  pair's syndromes. **s6 section 1 (elementary, PROVED):** if the pair is
  B-rational, every aligned slope Z = -a_r/b_r lies in B. Hence a genuinely
  F-valued (non-B) bad slope requires EITHER (i) F-valued words, OR (ii) the
  pole mechanism.
- **ext_pole_floor (PROVED):** the pole mechanism converts a base list of size
  L into genuinely-F-valued witness lines, N(L) = ceil(L(|F|-|B|)/(|F|-|B|+kL));
  it BINDS on non-generating rows (F \ B nonempty). This is the scope of
  ext_pole_floor: F-valued witness LINES arising from a base list.
- The extension-fiber incidence variety X/B and its reduced HORIZONTAL
  components are the geometric carriers of these aligned slopes over a row.

## The bridge (prove, or refute with an S9 obstruction)
> **Claim (ef_fullfield_bridge):** every reduced horizontal component C of X
> with trivial G-stabilizer, satisfying noncontainment, and not confined to a
> proper intermediate field, is a genuinely-F-valued witness line arising from
> a base list — hence is covered by ext_pole_floor and meets the pole divisor D.

Equivalently: the two s6 sources of genuinely-F obstructions (F-valued words /
pole mechanism, and subfield confinement) EXHAUST the genuinely-F horizontal
components. There is no third "genuinely-F obstruction of non-pole, non-confined
type" -- the object the s6 sketch names F1 and flags as the honest place a
Lean-ledger case split "would first fail to typecheck," where a new ledger
column S9 would otherwise appear.

## The ask
- **(A) Close the typecheck:** prove every trivial-stabilizer noncontainment
  non-tower horizontal component is a base-list F-valued witness line (so N(L) /
  ext_pole_floor applies). The likely lever: the pencil is LINEAR with
  B-defined coefficients (s6 section 1), so a genuinely-F component's slope
  Z = -a_r/b_r being outside B forces F-valued word data of exactly the
  base-list form the pole floor consumes -- i.e. the linear-pencil structure
  leaves no room for a non-pole, non-confined genuinely-F fiber. Make this
  precise. This discharges ef_fullfield_bridge -> ef_pole_free_cycle_exclusion
  -> ef_full_orbit_pole_forcing -> ef_ru -> f1_full_field_pole_forcing.
- **(B) Exhibit the S9 obstruction:** a genuinely-F horizontal component with
  trivial stabilizer, satisfying noncontainment, not tower-confined, and NOT a
  base-list witness line covered by N(L). This is a new paid ledger class (S9)
  and reshapes the extension classification -- decisive either way.
- **(C) Conditional:** the typecheck modulo a clean stated hypothesis on the
  pencil data (e.g. a non-degeneracy of the linear coefficients a(l), b(l)).

Note: on GENERATING rows (E = B) the bridge is trivial (G = 1, no F \ B); the
whole question lives on non-generating intermediate rows, which admissibility
forces to be tiny at the base (q_gen < 2^128). That smallness may itself be the
lever that closes (A).
