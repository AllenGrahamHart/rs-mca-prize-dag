# f_support_lattice proof

Each descent step is indexed by a sparse-dual support and replaces the current
state by subproblems determined by closed sets generated from those supports:
unions, closures under the trace dependence relation, corresponding subflats,
and residual domains.

If two branches reach the same closed set with the same residual degree and
flat data, their future recursive calls are identical. Memoizing by this
closed-set state therefore identifies the binary descent tree with a directed
walk on the support lattice generated by the sparse supports.

Along any strict chain, either the flat dimension drops or the locator degree
drops. Thus the chain length is at most the initial dimension plus the initial
degree drop budget. The number of memoized states is bounded by the number of
closed sets in the generated support lattice, up to the recorded bounded
weight-label factor.

This proves the accounting identity: the descent tree size is controlled by
the closed-set count. It does not prove a uniform polynomial closed-set bound;
that family-specific counting problem remains in the termination nodes.
