# Proof

The G2 node is stated on each contributor's agreement set `S`. Its periodic
branch sends that same set to the E22 staircase machinery. The proved
cross-scale canonical form uses the support `R` itself as its canonical object,
and `petal_g3_profile_conversion_identity` counts those supports on the
size-`A` slice. Thus the handoff has

```text
R=S,  A=|S|>=k+sigma>=k.
```

Suppose two degree-`<k` polynomials `P` and `Q` are both charged to the same
canonical support `R` for the same received word `U`. At every point of `R`,

```text
P(x)=U(x)=Q(x).
```

Therefore `P-Q` has at least `|R|>=k` distinct roots. Since `P-Q` has degree
less than `k`, it is zero, so `P=Q`.

This proves codeword injectivity within one full canonical support. The
support-only no-go `petal_g3_subk_support_only_no_go` does not contradict this:
it applies only when a proposed datum retains fewer than `k` agreement points.
It remains a guard against replacing the actual full support by a partial one.
