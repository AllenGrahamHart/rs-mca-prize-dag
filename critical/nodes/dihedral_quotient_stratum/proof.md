# dihedral_quotient_stratum proof

Let

```text
P_m = { p : p(X) = X^(2m) p(X^(-1)) }.
```

A polynomial in `P_m` is determined by the `m+1` symmetric coefficient
pairs, so `dim P_m = m+1`. Equivalently,

```text
P_m = X^m F[X + X^(-1)]_{<= m}.
```

Indeed, the expressions `X^m (X + X^(-1))^i`, `0 <= i <= m`, span exactly
the self-reciprocal degree-`2m` polynomials.

The flat is gcd-trivial on a multiplicative domain `H`: the member `X^m` has
no zero on `H`, so no domain point is a common root of all members.

It is not contained in a nontrivial multiplicative pullback `X^e g(X^M)`.
Such a pullback uses only exponents congruent to one residue modulo `M`.
But `P_m` contains monomial supports at adjacent exponents, for instance from
`X^m` and `X^m(X+X^(-1))`, so no `M > 1` can contain the whole flat.

Now remove the fixed points `±1`. The inverse-pair orbits have count
`(n-2)/2`. For any choice of `m` inverse-pair orbits represented by elements
`a`, the product

```text
prod_a (X^2 - (a + a^(-1)) X + 1)
```

lies in `P_m` and vanishes exactly on those selected inverse pairs. This
gives

```text
binom((n-2)/2, m)
```

members of `D_j` in the even-support convention `j = 2m`.

For each inverse pair `{a,a^(-1)}`, evaluation traces on `P_m` satisfy

```text
p(a) = a^(2m) p(a^(-1)).
```

Thus the two trace coordinates are linearly dependent; each inverse pair
supplies a weight-2 dual word.

The Chebyshev coordinate

```text
y = X + X^(-1)
```

identifies inverse-pair orbits. Since

```text
p(x) = x^m h(x + x^(-1))
```

with `deg h <= m`, evaluation on inverse-pair orbits is a nonzero diagonal
rescaling of degree-`<=m` polynomial evaluation in `y`. Distinct inverse
pairs give distinct `y` values, so the trace matroid is the rank-`m+1` MDS
matroid of Reed-Solomon evaluation in the Chebyshev coordinate.

Finally, every selected collection of at most `m` inverse-pair supports gives
a closed set generated by the corresponding weight-2 dependencies. Hence the
sparse-support closed-set lattice has at least

```text
sum_{h <= m} binom((n-2)/2, h)
```

members. For `m = omega(1)` this is super-polynomial, while for fixed
dimension `d=m+1` it is bounded by `n^O(d)`.

This proves the structural dihedral/Chebyshev quotient stratum and justifies
enlarging the quotient taxonomy to forms of the shape
`X^e g(X^M + X^(-M))`.
