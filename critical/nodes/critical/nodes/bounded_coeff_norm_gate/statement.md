# bounded_coeff_norm_gate

- **status:** PROVED

## Statement

For bounded integer coefficients |c_a| <= C and omega of 2-power order M in F_p: P_c(omega) = 0 forces the cyclotomic/coset case (c_a = c_{a+M/2}, counted exactly) OR p | Res(X^{M/2}+1, Q_c) — the explicit cleared norm. Tower sections contain no opposite pairs, so every nonzero tower skew is norm-gated at every imposed odd exponent simultaneously. No bounded-coefficient escape. (Pro; replayed, F_97 example Res = 97^2 exact.)
