## Primal Target

In the same notation as
`F3_H3_CONIC_RATIONAL_CURVE_INTERFACE.md`, let

```text
R_U=P_U^H, R_V=P_V^H, R_W=P_W^H, R_Q=Q^H,
M=3(B-1).
```

The official conic span is generated by

```text
X^a R_U^b1 R_V^b2 R_W^b3 R_Q^(M-b1-b2-b3),

0 <= a < A,   0 <= b_i < B.
```

It lies in binary forms of degree

```text
L = (A-1) + 2HM = (A-1) + 6H(B-1).
```

The rank theorem needed by the exact-profile route is

```text
codim span <= 1847
```

on every official exact-profile row.

## Dual Form

Let `ell=(ell_0,...,ell_L)` be a coefficient functional on binary forms of
degree at most `L`.  For each boxed exponent vector `b=(b1,b2,b3)`, define

```text
G_b = R_U^b1 R_V^b2 R_W^b3 R_Q^(M-b1-b2-b3).
```

Then `ell` annihilates the conic span if and only if

```text
ell(X^a G_b)=0
```

for every `0 <= a < A` and every `0 <= b_i < B`.  In coefficient form, this is
the system of shifted convolution equations

```text
sum_j coeff_j(G_b) ell_{j+a} = 0.
```

Thus the exact official theorem can be restated as:

```text
DUAL-CONIC-ANNIHILATOR:
  The vector space of sequences ell_0,...,ell_L satisfying all B^3
  length-A shifted product-window equations has dimension at most 1847.
```

This is equivalent to the primal codimension target by the standard
rank-nullity pairing between a subspace and its annihilator.

