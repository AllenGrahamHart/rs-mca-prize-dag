# F3 h=3 conic binary-form rank target

Status: PROOF-TARGET REDUCTION, NOT `RC-RANK`.

This packet restates the exact-profile same-fiber conic rank route as a
binary-form codimension theorem.  It does not prove that theorem.

## Cleared Conic Span

On a same-fiber conic chart with common denominator `Q`, write

```text
U=P_U/Q,  V=P_V/Q,  W=P_W/Q.
```

After denominator clearing, the h=3 rank columns are

```text
X^a P_U^(H b_1) P_V^(H b_2) P_W^(H b_3)
    Q^(H(3(B-1)-b_1-b_2-b_3)),

0 <= a < A,   0 <= b_i < B.
```

The four quadratic binary forms `P_U,P_V,P_W,Q` satisfy one affine linear
relation on a same-fiber chart:

```text
P_U + P_V + P_W + alpha Q = 0.
```

In the pinned conic chart over `F_769`, these four quadratics have span rank
`3`, so the relation is the only linear relation among them.  Since the total
quadratic exponent is `3H(B-1)`, the cleared columns lie in

```text
H^0(P^1, O((A-1)+6H(B-1))).
```

Thus the exact-profile rank theorem can be targeted as:

```text
codim span <= Delta
```

inside this binary-form space.

## Official Tolerance

The exact-profile deficit budget proves that the official rows need only

```text

```

Equivalently, for every official exact-profile box it is enough to prove

```text
rank >= (L+1) - 1847,       L=(A-1)+6H(B-1).
```

The tight row is `s=13`.  The official exact-profile boxes have

```text

B = 34..22721.
```

This is the proof-level target suggested by the conic large-gap pilot: the
theorem should bound binary-form codimension, not assert automatic full
degree-space rank in every small box.
