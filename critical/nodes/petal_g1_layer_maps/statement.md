# Witness-exhaustive bounded sunflower atlas

- **status:** TARGET
- **consumer:** `petal_growth`

For every official row and every received word `U`, there is an explicitly
defined finite atlas `A_U` covering every top-band full-petal contributor such
that:

1. each chart is a fixed `(D0,R0)` sunflower auxiliary layer with the injective
   map `Psi` into degree-`<=d` polynomials and the exact auxiliary agreement
   `a=sigma+d+1-|R0|`;
2. each chart's petals are cosets of its printed scale subgroup, so the
   periodicity scale used by G2/G3/K4 is defined;
3. the total number of charts satisfies

   ```text
   |A_U| <= n^b1
   ```

   for a printed row-independent exponent `b1`, with `b1+b4<=B` in the
   primitive consumer budget;
4. every contributor belongs to at least one chart, so a fixed total order
   gives a first covering chart.

The chart-count clause is aggregate and load-bearing. A bound on the number
of charts containing one codeword is not a substitute.
