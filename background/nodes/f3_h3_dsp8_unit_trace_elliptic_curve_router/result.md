# Result

The generic DSP8 trace fiber is elliptic, not conic. Its exact projective
model is

```text
X^2Y+XY^2-sigma XYZ+Z^3=0,
```

with only the nodal degeneracy `sigma^3=27`. The raw curve-pair count is the
existing split-pencil count `J=4K`, so no constant is lost. A closing theorem
must control subgroup points and the retained target/quotient correlation on
this elliptic family; rational conic parameterization is a fenced route.
