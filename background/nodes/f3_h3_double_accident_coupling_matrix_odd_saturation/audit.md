# Audit

The rectangle has many nonzero entries but low ideal-generation complexity.
Equation `(CM5)` is a route fence against treating the
`mr-min(m,r)` nonzero count as independent norm or index gain: after odd
localization, one row and one column already generate the rectangle.

Both directions of the partial-matching statement are needed. Quotient
uniqueness controls zeros along a fixed product row; product Sidonicity
controls zeros down a fixed quotient column. Neither statement alone gives
the bound in `(CM6)`.

The full-fiber matrix is an exact survivor object. Coarse candidate generation
still uses a low-distance three-product star and one nonzero quotient anchor;
it should not enumerate the full rectangle before candidate primes are known.
