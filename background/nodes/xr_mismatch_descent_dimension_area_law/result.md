# Result

The XR nongeneric recursion has an exact dimension budget, not only a depth
cap. Along one chain,

```text
sum(K_j+h)<=N_0-(h+1),       sum d_j<=K_0-1.
```

Thus long chains are necessarily low-dimensional, while high-dimensional
states occupy only the printed bounded number of levels. The remaining XR
task can now seek a branch or slope payment weighted by this area law instead
of paying the same worst-case dimension at every depth.
