# Audit

1. The verifier checks all 70 maximal minors, not a sampled MDS condition.
2. Euclidean self-duality is stronger than the weighted self-duality available
   in the endpoint branch.
3. Coordinate multipliers in a GRS code do not change its projective columns;
   row operations only apply a projective transformation.
4. A twisted cubic's three-dimensional quadratic ideal has two linear
   Hilbert--Burch syzygies. The exact full-rank product matrix excludes this
   projectively invariant condition.
5. The matrix is the published `C_11,8,8` fixture of Betsumiya--Georgiou--
   Gulliver--Harada--Koukouvinos; all properties used here are independently
   replayed, so the proof does not depend on its classification provenance.
