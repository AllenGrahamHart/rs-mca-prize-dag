# Audit

1. The example satisfies both primary zeros and the nonzero next coefficient.
2. The secondary check replays `LT/a_16`, not an unrelated square series.
3. The normalized square root has degree four, below the allowed degree five.
4. Splitting and squarefreeness are checked explicitly.
5. Nonparity is invariant under common scaling; the root set is not closed
   under negation.
6. The example is deliberately not promoted against the official theorem:
   its root orders do not divide 56 and its square classes are mixed.
7. The result forbids a proof using only gap/differential identities. It does
   not justify further raw subgroup enumeration.
