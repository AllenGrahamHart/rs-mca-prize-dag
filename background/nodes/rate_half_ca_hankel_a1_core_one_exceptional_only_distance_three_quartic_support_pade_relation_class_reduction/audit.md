# Audit

1. Relations are taken over `F(U)`, not only at one specialized orbit
   coordinate. This is exactly the meaning of an identically zero circuit.
2. A primitive pair is unique only as a rational relation. Polynomial
   multiples may give a larger matrix kernel but do not create new classes.
3. The class-intersection bound uses `AB'-A'B`, so it remains valid when
   either numerator or denominator has degree below `t`.
4. `F_R` cannot vanish identically because `I` and `Q_U` are coprime over
   `F(U)` and `t<e`.
5. Distinct classes may intersect in `2t` points. The shadow uses subsets of
   size `2t+1`, not size `2t`.
6. The official class thresholds use the exact circuit lower bound, not the
   rounded percentages printed for orientation.
7. The class-size target is a sufficient closure condition. A sharper
   aggregate bound in `(QPRC4)` may close without a uniform maximum.
