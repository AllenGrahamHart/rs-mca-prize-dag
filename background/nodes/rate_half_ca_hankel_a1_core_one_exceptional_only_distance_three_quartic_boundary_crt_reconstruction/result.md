# Result

The quartic-valued curve is no longer an object that must first be recovered
from all external slope cofactors. The exceptional pairs define one CRT
label `delta`, the first-jet sum divides by `z-delta`, and a known subgroup
derivative factor reconstructs `Omega` coefficientwise modulo `A`.

A valid design forces the generic degree-`<2e` remainder to collapse to
degree at most four. This is an exact, low-memory rejection gate with a
compact certificate. The remaining theorem must explain that collapse well
enough to force a line decomposition or a contradiction; this node does not
assert either outcome.
