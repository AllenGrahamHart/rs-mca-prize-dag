# Audit

- The construction uses all blocks as sets, so repeated projective locator
  classes cannot hide in the rank calculation.
- Translation orbits prove replication exactly; it is not inferred from an
  average count alone.
- Quadratic rank is decomposed into the eleven distinct cyclic characters.
  The three-block partition orbit adds the missing direction only in
  character five.
- Complement span is calculated after exact division by `z^15-1`; Fourier
  inversion makes the nine printed coefficient axes an equality, not an
  upper bound.
- The universal residue rejection uses only `deg I=5` and `I(0)!=0`. Split,
  squarefree, and disjoint-root assumptions would only narrow the class.
- This is a route fence. It does not weaken the official rank-three residue
  gate and does not claim that the cyclic design lifts to the Hankel chart.
