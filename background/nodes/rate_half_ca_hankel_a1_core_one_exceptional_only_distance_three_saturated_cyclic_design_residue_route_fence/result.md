# Result

The saturated generic branch cannot be closed from biregularity, quadratic
rank `3e+1`, and complement span `e+4` alone: one exact cyclic `e=5` design
passes all three. It fails every possible calibrated rank-three reduction
modulo an internal locator with nonzero roots. Future theorem and compute
work must retain the actual modulo-`I` residue matrix, not replace it by the
unreduced complement-span bound.
