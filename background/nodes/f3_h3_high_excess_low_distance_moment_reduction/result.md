# Result

The selected `P>=33` tail is now a concrete weighted collision moment:

```text
M_33=sum_(P(t)>=33)(2N_4(t)+N_6(t))R(t).
```

Proving its exact bound `(HLM3)`, conservatively about `18.92n^2`, closes
C36'. This exposes two separate continuations: apply the existing
two-variable routers to the distance-four term, and develop an algebraic
six-term estimate for the distance-six term. Prime enumeration is one way to
certify this moment at fixed order, but no longer the only stated interface.
