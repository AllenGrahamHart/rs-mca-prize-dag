# Audit

1. The normalization uses common subgroup scaling and preserves the primary
   gap; it does not choose one of the six completion pairs.
2. `F_H` is the even coefficient and `G_H` is the odd coefficient after
   dividing by `S`. The division is valid only in the exactly-one-antipodal
   branch.
3. The sign-free torsion circuit starts at squared roots, so it has 39
   updates, not 40.
4. Both terminal equations are required. Product torsion alone does not put
   each complementary root in `mu_N`.
5. `Res_X(F_H,G_H)=0` is only a first gate; the same common root must pass the
   trace circuit and square/distinctness conditions.
6. No bounded-row calculation is cited as proof of emptiness. The theorem is
   an exact reduction only.
