# Audit

The distance-four cap is subtracted fiber by fiber before quotient weighting.
Using only its global edge count would not justify `(H6M2)`.

The antipodal branch is the worst branch only after its norm-one vector is
retained in the centroid identity. Using the all-norm-three centroid there
would discard a real saving. Antipodal-free fibers still have a stronger
distance-six lower bound.

The ratio `8/21` is sharp for the integer inequalities, not asserted to be
attained by an actual official row.

`M_6,33` retains both the `P>=33` filter and `R(t)` pointwise. The selected
target is not an unweighted distance-six census.
