# Result

The exceptional high-distance tail now has two structurally different
regions. Through `h=e+1`, there is one canonical minimal support and every
ordinary fiber attaining it is an internal fiber with a disjoint
`(h-1)`-point cancellation set. Only the upper region can require multiple
minimal-support leaders.

The next lower-region theorem should work with the single Forney pair
`(B_T,F)` rather than enumerate support subsets.
