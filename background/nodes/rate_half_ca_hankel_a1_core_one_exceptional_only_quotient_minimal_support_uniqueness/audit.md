# Audit

The uniqueness threshold comes from the full moment-column dimension
`2r+1`, not from the quotient dimension alone. At `h=e+1`, the union in the
two-leader comparison has at most `4e+2` columns, still below `2r+1=4e+3`.
At `h=e+2`, this argument can involve `4e+4` columns and no uniqueness is
claimed.

The clean locator has `r` roots, so `|H_z|=h` implies `a_z=r-h`; the
off-by-one cancellation count is then `h-1` because the canonical support on
`R_A union T` starts with `r+h-1` nonzero positions before cancellation.

This result forbids a fleet of alternative minimal supports in the lower
high-distance interval. It does not make the canonical support easy to find
or prove that it satisfies the remaining reciprocal and splitting gates.
