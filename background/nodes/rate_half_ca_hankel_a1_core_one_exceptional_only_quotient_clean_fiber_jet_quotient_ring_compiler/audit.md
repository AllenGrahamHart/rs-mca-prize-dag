# Audit

1. The coefficient field and the reciprocal-kernel biform are both commonly
   denoted `F`; `K_gamma` explicitly distinguishes the quotient algebra.
2. Unit claims follow from squarefreeness and pointwise nonvanishing on every
   root, so they remain valid if `f` is not irreducible.
3. Exponent reduction uses `f | R_X | (Y^M-1)`, not a claim that `f` equals
   the full domain locator.
4. `w_Y` means the formal derivative of the canonical degree-`<r`
   representative. That representative equals the actual fiber polynomial.
5. The theorem removes root enumeration and huge exponents, not the need for
   compressed arithmetic at degree `r=2^39-1`.
