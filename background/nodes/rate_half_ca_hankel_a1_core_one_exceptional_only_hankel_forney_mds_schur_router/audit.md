# Audit

1. In systematic form, MDS is used twice: `B` is invertible and its first
   row has no zero coordinate.
2. The square products are separated from the cross products by the first
   `e` coordinates, so their dimensions add.
3. The upper bound comes from the nonzero Forney residue functional; the
   MDS product argument alone only proves the lower bound.
4. Complementary dependence uses nonzero diagonal weights. Squared minors
   avoid orientation signs.
5. The router is exhaustive but neither output is declared impossible.
