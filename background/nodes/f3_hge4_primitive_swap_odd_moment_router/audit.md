# Audit

1. The support action is multiplicative scaling; exponent translation is
   used only in the finite verifier.
2. `Q=-P` follows from the already-proved swap stabilizer classification.
3. The constant coefficient is checked separately; it is what excludes even
   `h`.
4. Only odd indices below `h` vanish. The top odd elementary symmetric
   function is nonzero and supplies the shift.
5. Newton equivalence uses `p>h`, valid because official `p>=n^2`.
6. Global sign reversal exchanges the two orientations and does not create a
   second union.
7. The candidate binomial counts antipodal unions, not sign vectors or
   ordered pairs.
8. Primitive common stabilizer remains an explicit converse hypothesis.
