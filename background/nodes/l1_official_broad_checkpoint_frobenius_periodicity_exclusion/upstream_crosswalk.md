# Upstream crosswalk - broad-checkpoint periodicity exclusion

In primitive shift-pair language, the signed support is a balanced
`{0,+1,-1}` base-field word whose Fourier transform vanishes on the first
`p` subgroup characters. Base-field normalization closes that zero set under
characteristic Frobenius. On the seven broad official rows the residual
characters all lie in an even-index subgroup, forcing an impossible even
period on each sign support.

This is a constants-free exact exclusion, not a second-moment estimate. A
portable handoff should retain the seven-row orbit table and route all further
minimum-width work to the nine rows `n=m(p+1)`.
