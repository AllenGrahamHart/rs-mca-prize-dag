# Audit

The following points are load-bearing.

1. The signed roots, not the base exponents, enter Newton's identities.
2. `1593` is the inverse of `9` modulo `2048`; normalization is unique.
3. Reduced support forbids antipodal roots, so squaring loses no root.
4. The norm identity is `-F(X)F(-X)=G(X^2)`; the minus sign is required.
5. The converse uses `rho=A(y)^(-1)`, not an arbitrary square root of `y`.
6. Divisibility by the separable `Y^1024-1` supplies nine distinct roots.
7. Empty characteristic zero implies only a finite characteristic
   obstruction. It does not identify or exclude those characteristics.

The proposed contributor endpoint is a checked identity
`Delta=sum H_jR_j`, not a point sample or an unsupported Groebner claim.
