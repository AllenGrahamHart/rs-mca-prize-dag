# Result

The deepest WCL slot no longer requires a search over signed nine-element
supports. Up to its unique common dilation, it is exactly the problem of
classifying monic quartics `A` for which

```text
Y A(Y)^2-1 divides Y^1024-1.
```

The nine remainder coefficients form a four-variable integer ideal with no
characteristic-zero points. A single checked integer Nullstellensatz
certificate therefore reduces all possible survivors to finitely many
characteristics.
