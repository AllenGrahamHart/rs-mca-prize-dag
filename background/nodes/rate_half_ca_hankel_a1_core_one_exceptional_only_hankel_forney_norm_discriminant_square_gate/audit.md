# Audit

1. The sign in `(4)` is `(-1)^e` because
   `(2e)(2e-1)/2` has parity `e`.
2. The hyperbolic determinant condition contributes the same `(-1)^e`, so
   the two signs cancel in `(5)`.
3. All three factors of `C=q_1Phi/B_T` are units modulo `A`; every resultant
   in `(HNS1)` is nonzero.
4. The square condition does not require the exceptional code to be MDS.
5. No square root is canonically chosen; a positive certificate may print
   any nonzero witness whose square is `Xi_A`.
