# Audit

1. `C` is the unused residual square, not a member of the complementary
   pair `{B,D}`.
2. Ordered pairs `(A,C)` are required: reversing them changes the repeated
   completion square and generally changes the invariant class.
3. The divided quartic `H` removes the forbidden diagonal `C=A`; using
   `D_*(C)` would add four unsupported tests.
4. The clearing exponent is six because the norm has maximum `P`-degree
   six. Since `e_4!=0`, clearing denominators preserves the zero set.
5. The exact prefactor is `e_4^36`, which is nonzero.
6. The resultant is a compiler for the coupling only. Canonical span and the
   official coefficient gaps remain separate obligations.
