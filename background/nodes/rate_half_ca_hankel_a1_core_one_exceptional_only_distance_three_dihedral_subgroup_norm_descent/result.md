# Result

The remaining dihedral split perfect-power test no longer requires a direct
norm over all `N` smooth-domain points. Its exact input is an orbit resultant
with at most `N/2` nonfixed quotient roots and a degree-at-most-`r` value
polynomial. The constant-product quotient locator is recoverable from a
Dickson recurrence with an explicit zero-or-two-fixed-point correction.
External split slopes themselves are equivalently detected by splitting this
degree-`r` orbit polynomial over the quotient set.
