# Result

The HGE4 residual can be attacked as a quadratic orbit census:

```text
sum_(h=4)^H_max O_h^prim<=14n^2.
```

Each `O_h^prim` counts canonical scaling orbits of ordered primitive top
shift pairs. A left-anchored ordered scanner returns exactly `hO_h^prim`, so
it must divide by `h` or apply one further rotation canonicalization. This is
the preferred finite certifier currency. A plain Q
maximum-fiber bound is separated from this route unless it proves primitive
emptiness; the needed nonzero theorem is an averaged falling-factorial or
orbit estimate.
