# Lineage

The terminal shift-pair theorem reduced every oriented difference to a
constant-weight residual code but used only Plotkin and the fixed-side
intersection cutoff. The puncture compiler then paid `K-t<=2` per fixed
difference without aggregating difference types.

This node observes that at codimension `c=K-t`, the same low-core inequality
makes every residual own disjoint `c`-subsets. Charging those subsets before
the energy sum gives an aggregate near-`K` payment and removes multiple
widths without computation.
