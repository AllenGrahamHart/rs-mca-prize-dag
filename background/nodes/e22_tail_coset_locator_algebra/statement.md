# e22_tail_coset_locator_algebra

- **status:** PROVED
- **closure:** proof

## Statement

If `M | n`, `B` is a tail of size `|B| < M`, and `H` is a set of quotient
fibers of `x -> x^M` in the multiplicative subgroup domain, then the locator
of `B` together with the full fibers over `H` is `L_B(X)G(X^M)`.

If `M > t`, its top `t` subleading coefficients depend only on `B` and the
number of selected fibers, not on which fibers are selected.

## Falsifier

A tail-plus-full-coset support whose locator is not `L_B(X)G(X^M)`, or whose
top `t` coefficients depend on the selected quotient roots despite `M > t`.
