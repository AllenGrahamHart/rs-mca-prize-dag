# Result

The sharp endpoint is reduced on the Hankel side to two explicit weighted
self-dual split-polynomial frame classes. Their columns encode exceptional
incidence blocks, and their row replications are flat except for at most one
opposite pair of unit deviations.

An exact `e=3,F_101` flat frame survives all of these axioms, so the next
exclusion must use official Forney weights, domain placement, or scale.
