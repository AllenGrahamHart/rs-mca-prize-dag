# Audit

- Used the literal Hankel indexing `(M_1)_(i,j)=h^(1)_(i+j)`; the three
  shifts reach moment indices `2r-2`, `2r-1`, and `2r`, respectively.
- Derived all three scalar conditions from the proved kernel-plane pairings;
  none is imposed as an additional hypothesis.
- Kept the contracted source weights abstract. Contraction and the local
  parameter change preserve a finite source sum but generally change the
  original error weights.
- Used `A(x)^2` as a polynomial product, so the source identity is valid in
  characteristic two as well as odd characteristic.
- Deleting roots from the source sum uses only `A(x)=0`; it does not claim
  that source weights vanish there.
- The moment pin is necessary, not sufficient: lower Hankel coefficients,
  reciprocal degree boxes, and splitting remain open.
