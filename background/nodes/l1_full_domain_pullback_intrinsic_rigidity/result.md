# Result

Every monic polynomial whose equal complete fibers partition a smooth
multiplicative coset is `X^s+c`; there are no non-intrinsic full-domain
pullbacks. Fully fiberwise exact supports are therefore exact-periodic and
already paid, including the former sub-Johnson sector. The live general-
pullback problem begins only with incomplete domain coverage, partial-fiber
agreement, or lack of a whole-petal anchor.
