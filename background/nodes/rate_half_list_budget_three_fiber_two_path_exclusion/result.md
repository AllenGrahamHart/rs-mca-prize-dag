# Result

The direct equal-complete-fiber residue in both path-plus-singleton chambers
is now empty for common fiber size two. Combined with the preceding
Vandermonde theorem, every common monomial fiber size `m>=2` is excluded in
the path branch.

This removes one exact small-fiber case. It does not close the budget-three
branch: fiber size one carries no quotient compression, and mixed-map,
partial-fiber, and primitive path constructions remain. The four-cycle
fiber-two branch also remains live.
