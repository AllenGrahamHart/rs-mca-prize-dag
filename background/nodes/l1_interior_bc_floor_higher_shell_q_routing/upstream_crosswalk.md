# Upstream crosswalk - interior BC floor to higher-shell Q

## Paper D object

`prop:capg-census-floor(b)` constructs a level-`m'` identity-prefix fiber and
counts `binom(m',m)` size-`m` supports from each exact level-`m'` codeword.
That is an unavoidable floor for Paper D's unguarded support census and its
MCA/rank-one uses.

## L1 correction

For the ordinary-list exact-shell ledger, every one of those proper
sub-supports fails the complement gcd guard.  The codeword belongs once to
the level-`m'` boundary-Q bucket.  Therefore `M_B^disc(d_1)` is not an
unavoidable lower model for the guarded level-`m` L1 interior cell.

## Remaining input

consumers.  A list-side compiler should expose a separate
`BC_exact_guarded` numerator after higher-shell ray deletion.  Bounding that
residual, and coalescing its owners with Q across levels, remains open.
