# Audit

The leader supports live in all of `D_res\R_A`. The residual domain has
`8e+7` points and `R_A` has `2e`, so the universe size is `6e+7`, not the
`6e+6` roots of the split factor complementary to `R_A`; the unique omitted
residual row remains available to a quotient support.

The intersection cap uses a relation on `R_A union T union T'`. The first
dependent cardinality is `2r+2`; using `2r+1` would lose one and corrupt the
official cutoffs.

The Johnson bound controls the number of leaders, not the number of possible
supports from which those leaders might be selected. It therefore supports a
finite symbolic leader classification but does not authorize combinatorial
enumeration of `h`-subsets.
