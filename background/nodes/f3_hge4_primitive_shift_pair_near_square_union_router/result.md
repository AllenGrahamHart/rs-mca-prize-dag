# Result

HGE4 primitive top shift pairs can be generated without enumerating their
left/right partitions. Enumerate a `2h`-support union, compute its monic
locator, recover the unique monic square root at infinity, and retain it only
when the remainder is a nonzero scalar square and the reconstructed pair is
primitive.

For an anchored scan this removes the exact factor
`binom(2h-1,h-1)` while preserving the exact output identity

```text
A_h^union/h=O_h^prim.
```

Future fixed-row contributor certifiers should use this union currency. The
uniform aggregate remains open.
