# Claim contract

## Proved

- `(SL1)` exactly characterizes arbitrary support-wise MCA-bad slopes.
- Syndrome-pair enumeration is exhaustive modulo code translation.
- The printed `F_7` toy pair has all seven finite slopes bad, with one
  explicit moving two-column secant support per slope.

## Not proved

- The toy witness lifts to an official smooth-domain row.
- A raw syndrome-pair enumeration is computationally possible at official
  redundancy `2^40`.
- The old polynomial locator-fiber census tests arbitrary received lines.
- Any current candidate for the official adjacent crossing is false.

## Falsifier

A received line and support whose direct MCA status disagrees with `(SL1)`,
or a slope at which the printed toy pair is not MCA-bad.
