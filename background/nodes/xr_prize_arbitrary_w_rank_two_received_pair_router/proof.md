# Proof

For a selected block at slope `gamma_i`, write

```text
b+gamma_i q+w_i=0       on A_i,       w_i in W.       (1)
```

The trade row `lambda_i` is supported in `A_i` and annihilates `W`, so
pairing `(1)` with it gives

```text
<lambda_i,b+gamma_i q>_X=0.                          (2)
```

Write `lambda_i=c_iF+d_iG`. Expanding `(2)` gives

```text
c_i<F,b>+d_i<G,b>+
gamma_i c_i<F,q>+gamma_i d_i<G,q>=0,                 (3)
```

which is `(IR2)`.

For five blocks, the Segre router says that the five coefficient columns have
rank four. Their common annihilator is zero, so `(IR2)` forces `theta=0` and
proves `(IR3)`.

For four blocks, the Segre columns have rank three. Their annihilator is the
one-dimensional line generated by the canonical hyperplane vector
`(A,B,C,D)`, proving `(IR5)`. If `eta=0`, all four pairings vanish. If
`eta!=0`, the interaction matrix in `(IR6)` is

```text
eta ((A,C),(B,D)),
```

whose determinant is `eta^2(AD-BC)`. The Mobius row-class map is

```text
[c:d]=[B+D gamma:-(A+C gamma)].                      (4)
```

It is nonconstant exactly when the two linear forms are not proportional,
equivalently `AD-BC!=0`. Thus the matrix is invertible in the nonzero branch.

Finally, reversing expansion `(3)` proves that `(IR3)` or `(IR5)` makes every
trade-row pairing `(2)` vanish. This converse concerns those parity rows only;
the parent selected-block certificates remain necessary for actual agreement.
No property of `W` beyond row orthogonality was used. QED.
