# Audit

1. The relation `v^TM_1q_i=0` uses every coefficient recurrence, including
   the separate top equation `M_1q_e=0`.
2. `v^TM_1v!=0` proves both `v notin W_q` and restricted rank one.
3. Orthogonal-complement dimensions account for the one-dimensional
   intersection of `W_q` with each ambient radical; using the nondegenerate
   formula would be off by one.
4. The theorem identifies the regular quotient line but does not choose a
   normalization for its nonzero scalar.
