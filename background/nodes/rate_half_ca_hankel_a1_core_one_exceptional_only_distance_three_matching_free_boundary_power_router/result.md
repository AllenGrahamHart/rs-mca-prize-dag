# Result

The distance-three boundary-power gate no longer requires enumeration of
perfect matchings. The `2e` exceptional derivative values, after taking the
`e`-th power, must form a centrally symmetric multiset; this is one exact
even-polynomial test. The three boundary values must have one common
`e`-th power.

The result uses no Modal resource. It gives a compact algebraic target for a
future support classifier, but it does not close the distance-three chart.
