# Audit

- The node point is counted separately because the line parameterization
  removes its double intersection. The two tangent parameters map back to it
  and are excluded; it lies in the subgroup only for `c=1`.
- Quotient classes are load-bearing. The correct relation is
  `[1+theta]=[c][theta]^2`; replacing `[c]` by its inverse is false.
- `N_sigma` counts ordered decorated curve points, not unordered triples.
  This is the currency used by `G=J=4K`.
- `(NTP5)` deliberately discards richness, signed disjointness, and target
  collisions. It is a valid ceiling but its constant cannot be consumed.
- The leading-coefficient comparison uses the class weight `17`, not an
  unweighted sum or the cheaper antipodal-free coefficient `10`.
