# Result

Candidate-prime support for C36' can be generated from one structured
quotient-algebra presentation instead of a product over individual quotient
lifts. At cutoff 18, the `X` Fitting integer detects the complete rich-on-
quotient-support locus and the smaller `Y` integer detects exactly the
double-accident locus. The latter has the same official-prime support as the
existing `f_n` router but does not inherit its stronger `p^Y_18` valuation.
For support, the preferred output is the still smaller scalar annihilator,
equivalently the largest Smith invariant or scalar elimination generator; it
divides the full Fitting order and has the same prime support. The remaining
algorithmic problem is a succinct Smith/subresultant method for this
structured annihilator, not a dense maximal-minor computation.
