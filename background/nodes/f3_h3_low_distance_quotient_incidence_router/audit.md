# Audit

The exact identity `(LQI3)` uses canonical orientations. Without them, a raw
root tuple can represent the same unordered edge under endpoint and internal
root exchanges.

`R(t)` counts ordered quotient representations, so `z` is not quotiented by
exchanging `z,w`.

The distance-four systems are covers with one selected certificate per edge.
Dropping that certificate gives an overcount, which remains valid for an upper
bound but is no longer the exact identity `(LQI3)`.

The requirement `P(t)>=33` is retained. Removing it gives a stronger incidence
problem and may restore large low-excess strata that the `E=14` theorem has
already paid.

The variable counts describe the algebraic interface, not a point-count
theorem. No independence or random-density assumption is consumed.
