# Result

The remaining quartic boundary condition is linear in only `e` nonzero
external labels. A concrete matrix with `(2e-5)(e-1)` rows is determined
before the external quartics are reconstructed; full column rank or a
coloop column excludes the packet exactly. Deterministic controls are
full rank from `e=4` through the first official-shaped `e=7` analogue.

The next theorem target is correspondingly precise: prove full rank, or
torus-kernel emptiness, for every official admissible pair-Lagrange packet.
This reduction does not itself prove that theorem.
