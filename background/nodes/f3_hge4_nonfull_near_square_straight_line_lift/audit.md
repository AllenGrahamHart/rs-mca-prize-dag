# Audit

1. The characteristic-zero descent is reproduced in the proof because the
   legacy x24 source path is absent from this worktree.
2. Squared supports remain disjoint at every descent because both original
   supports are antipodally closed.
3. The selector omits `s_0`; a nonzero constant term does not distinguish a
   full fiber.
4. Selector variables add multiplicity and must never be used for counting.
5. The recurrence modulus has degree `2h`, not `h` as in the swap descent.
6. The terminal exponent is `n=2^s`, not `n/2`.
7. On official odd-characteristic rows, divisibility into squarefree
   `X^n-1` already forces `a!=0`.
8. One chart certificate closes only one fixed `(n,h,j)` cell unless a proved
   transfer combines it with the other charts and rows.
9. No elimination cost is inferred from cubic degree.
