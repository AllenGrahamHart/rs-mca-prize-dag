# Audit - L1 general polynomial-pullback interleaving descent

1. Monicity of `P` is used in the leading-term reduction.
2. Component degree caps retain the residue index `j`; they are not all
   silently rounded down to `k/s`.
3. Only complete `s`-point fibers use the Vandermonde equivalence.
4. Distinct fiber labels make the quotient evaluation domain valid.
5. Sparse complete-fiber coverage carries the exact kernel exponent `kappa`.
6. The heterogeneous component codes are enlarged to one common code only
   after the kernel charge.
7. The interleaving collapse is invoked only for `1<=L<q`.
8. `L^2<q` removes interleaving growth, not the kernel factor.
9. Full domain partition plus `k<|D|` proves `kappa=0` exactly.
10. No multiplicative, cyclic, or smooth-label hypothesis is inserted.

The next consumer theorem must bound the ordinary quotient list on the
realized label domain and handle partial-fiber boundaries or sparse coverage.
