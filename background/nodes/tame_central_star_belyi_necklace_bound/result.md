# Result

The central-star passport has no hidden map moduli. Its geometric cover
classes are bounded by the binary-necklace count

```text
N(c,e)=(1/c) sum_(d|gcd(c,e)) phi(d) binom(c/d,e/d).
```

This converts a tame polynomial-cover count into a finite combinatorial
count, without asserting that all necklaces are realizable.
