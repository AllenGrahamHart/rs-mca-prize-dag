# Upstream crosswalk - L1 decorated shift pairs

## Exact overlap

At scalar cofactor, the compiler is exactly upstream's locator-Q exact second
moment: common support `G`, disjoint equal-degree tails `A,B`, and
`deg(A-B)<=deg(A)-w-1`.

## New arbitrary-word decoration

Positive cofactor degree replaces `A-B` by `AQ_1-BQ_2`. This is the natural
second-moment object for the arbitrary-word list-interior term. In the
primitive range `e<=w`, the local uniqueness theorem removes any cofactor
multiplicity after `(A,B)` is fixed.

## Portable target

An upstream list-interior compiler can consume `(DS1)--(DS10)` directly. It
must then count primitive split support pairs, pay common cofactor gcds as
tangent/common-factor cells, and handle `e>w` separately. Existing
degree-three DSP8 and top-shift-pair theorems are special low-decoration
instances, not a uniform proof of this compiler's support census.
