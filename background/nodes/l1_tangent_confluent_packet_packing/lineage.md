# Lineage

`l1_tangent_hasse_root_pinning_census` paid one fixed owner but left the raw
sum over feasible owners open.  This node performs the requested collective
payment by packing the Hermite conditions themselves.  It is compatible with
`l1_tangent_double_root_descent_to_primitive_shell`: packet packing is a
global precharge, while owner descent identifies the residual primitive
object after that precharge.
