# Upstream crosswalk

Paper D's primitive split-pencil programme separates common-factor/tangent
owners from the primitive branch but does not print a collective owner
ceiling.  The fixed-owner estimate `q^max(k-2r,0)` is too expensive when
summed over all `binom(n,r)` possible divisors.

The present packet inequality is a portable replacement for that raw sum.
It uses only the exact agreement locator and `D^2|U-P`, so it can be inserted
before either the BC split-pencil census or the primitive shift-pair ledger.
For each degree cutoff `r0`, the minimum in `(HP3)` is an exact integer
precharge. In the balanced band `(HP8)` additionally refines the existing
complement packing by exchanging complement marks for tangent marks. It does
not prove Paper D's remaining primitive split-pencil or row-sharp Q
statements.
