# Audit

1. The recurrence is modulo `y^h`, so it determines the first `h`
   reciprocal coefficients; `P(0)=1` is supplied separately by scaling.
2. Characteristic greater than `4h` makes every recurrence denominator
   `j+3h` invertible. The weaker complement-third condition `p>m` alone does
   not state this, although official `p>=n^2` is much stronger.
3. The pair is ordered. Swapping it replaces `x` by `x^(-1)` and is counted
   separately unless the ordered records coincide, which `d!=0` excludes.
4. The injection is one-way. A root of `(LBO3)` may fail locator splitting,
   disjointness, exact ratio, or primitivity.
5. The `x=-1` deletion uses `h>1`. The toy `m=4,h=1` pair is outside scope
   and is a mutation guard against dropping that condition.
6. The `m-2` payment is per exact level and one width. It is not an aggregate
   bound for the remaining `e>=2` systems.
