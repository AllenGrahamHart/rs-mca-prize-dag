# Result

The clean C36' energy route should be compared with literature using

```text
E_x(H-1) <= (145/4)(n-1)^2+(2n-1)^2,
```

not by applying `145(n-1)^2/4` directly to the full shift. Relative to a
centered estimate `E_x(H-1)-n^4/p`, every official row has more than
`39.24n^2` of error allowance. This repairs the source normalization and
widens the constants budget, but does not prove the required estimate.
