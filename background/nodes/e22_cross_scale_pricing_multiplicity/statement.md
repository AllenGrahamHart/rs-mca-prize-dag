# e22_cross_scale_pricing_multiplicity

- **status:** PROVED
- **closure:** proof

## Statement

The quotient staircase pricing column counts the canonical support-scale
equivalence classes from `e22_cross_scale_support_canonical_form` with exactly
the declared multiplicity, so quotienting by cross-scale duplicates neither
overcounts nor undercounts E22 challengers.

This is proved from the dyadic minimal-scale selector and the
pricing-column compatibility theorem.

## Falsifier

A canonical cross-scale equivalence class whose contribution to the pricing
column has the wrong multiplicity.
