# Audit - PMA sigma-one dyadic near-coset owner

1. The owner uses the full source agreement set and is defined before every
   sunflower or defect chart.
2. The scale and coset are part of the owner. Canonical first match prevents
   one codeword near several cosets from receiving several charges.
3. The outside record has exactly `max(0,k-L+e)` points; reducing it by one
   can leave fewer than `k` interpolation points.
4. The owner is applied after exact periodicity, so `(COMBINED-COS)` adds
   disjoint classes and uses the sharper periodic count only once.
5. The all-scale estimate deliberately overcounts every dyadic scale and every
   defect value through `s_n=floor(n/(32 log_2 n))`. Its comparison with
   `Q_2` uses `2^7<15^2`, not floating entropy.
6. The index-two owner and reciprocal-quadratic subsumption are literal
   special cases, not analogies.
7. `QOWN_cosgrow` is distinct from low-boundary `QCLOSE`: an arbitrary
   low-defect error support need not contain a nearly complete large coset.
8. The finite/asymptotic firewall remains explicit.

The theorem does not pay the far-from-coset diffuse complement, defects above
the growing window, chart composition, or the general asymptotic
`GROW union RES` class.
