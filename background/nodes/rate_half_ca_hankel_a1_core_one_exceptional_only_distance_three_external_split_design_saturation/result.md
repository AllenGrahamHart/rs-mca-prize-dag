# Result

The external distance-three fibers form an exact biregular design, not a
near-saturated family. Exactly `6e+3` residual points outside the canonical
support occur in `e` of the `3e` external locators; the one remaining outside
row occurs in none. Equivalently, the product of the monic external locators
is exactly the `e`th power of the complement-domain polynomial.

This converts the external split condition into one compact power identity
plus block-level reconstruction. Any finite classifier can reject incidence
packets before Hankel elimination unless they satisfy this exact design.
