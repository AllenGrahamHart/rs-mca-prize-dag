# Result

The primitive cubic SP residue has no free scale. It is exactly a
determinant-one equal-trace pair of triples, with the decoration forcing
`q=rs` and the original target given by

```text
t=1+rs(r+s-sigma).
```

The live analytic object is therefore two subgroup-split quadratics coupled
to one affine quotient line. Generic all-pairs SP enumeration is fenced; a
proof must exploit this normalized trace/split correlation.
