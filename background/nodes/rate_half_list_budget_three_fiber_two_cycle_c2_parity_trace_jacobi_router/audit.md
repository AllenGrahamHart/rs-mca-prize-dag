# Audit

1. The parity parameter is `tau=r^2`; the cross-pair trace uses
   `chi=r+r^(-1)`.
2. The six pairs have three trace classes with multiplicity two, not two
   classes and not six independent tests.
3. `K(0)=0` is exactly `J=0` because the official characteristic exceeds
   three.
4. The cross-class product is needed to remove `r->-r`; either factor alone
   is orientation-dependent.
5. The Jacobi prefilter uses only the denominator primary equation and
   torsion. The c=2 invariant norm and secondary gate are later filters.
6. `tau=-1` is excluded, but non-pure parity values remain possible and must
   not be discarded without the shared norm result and branch replays.
7. The theorem assumes parity geometry; bounded evidence does not prove that
   every primary-plus-secondary denominator has it.
