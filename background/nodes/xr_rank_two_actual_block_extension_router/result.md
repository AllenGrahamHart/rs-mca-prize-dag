# Result

Per-row agreement-block realization is now exact. The selected trade row
pays one of `d+1-z_i` support-local checks; the other checks split into
`d-z_i` local and `tau_i` extension-sensitive conditions. After unique
degree-`<a` interpolation on the support, every actual block extension is a
`tau_i`-subset of one explicit external residual zero set, giving the exact
binomial count.

The remaining rank-two work is family-level support enumeration,
intersection compatibility, first-core ownership, and cross-core payment.
