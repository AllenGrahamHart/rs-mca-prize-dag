# Audit

1. The resultant includes the leading-coefficient factor `a^m`; omitting it
   would leave a false `m`-dependent trace formula.
2. Reversal multiplies each root value by `y_i^(r-1)`, producing the
   `q_0'/q_0` correction in `(QLT2)`.
3. All logarithmic derivatives are evaluated only at clean slopes, where
   `E`, `q_bar`, `q_0`, and every selected value of `W_vee` are nonzero.
4. The trace is valid for the split squarefree quotient algebra. No
   irreducibility of `f` is assumed.
5. The theorem produces a necessary scalar check, not a contradiction by
   constraint counting.
