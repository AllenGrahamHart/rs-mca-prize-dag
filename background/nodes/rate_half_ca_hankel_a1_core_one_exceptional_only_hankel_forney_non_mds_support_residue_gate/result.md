# Result

Every non-MDS annihilator certificate now carries a support-side scalar gate
linking its quotient factor to the Forney numerator and minimal support. The
test is a single top coefficient after reduction modulo `B_T`; it requires no
exceptional-root enumeration or maximal-minor search. An equivalent
exceptional-side formula is zero below the maximum quotient degree and equals
`Theta_2 lc(K)` at the sharp boundary, allowing either compressed side to be
used by a checker. Half-rank deficiency `d` upgrades the scalar to a
`d`-by-`d` matrix of cross-residue constraints.
