# Result

The first deleted-pair scalar rejection no longer needs the source trace
variable. According to `epsilon`, source torsion is `T_(2L)(x)=0` or
`U_(2L-1)(x)=0`; each of the six sign branches is then one exact common-root
test with `C_L^(1/4)(x)` and a printed polynomial `E_(j,s)(x)`.

After reduction modulo the primary Gegenbauer polynomial, every branch is a
degree-at-most-`L` gcd problem. A compact Bezout or subresultant certificate
can therefore prove rejection without enumerating source roots. Uniform
characteristic control for these six gcds remains open.
