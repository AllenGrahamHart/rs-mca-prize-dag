# Result

For arbitrary anchored polynomial pullbacks, a cap `Z` on agreements outside
fully agreed complete fibers yields an exact quotient threshold `h_Z`. The
strict Johnson side costs `q^kappa b^2` per map and is polynomial across all
tame fixed-source maps whenever `kappa` is bounded. The live partial-pullback
frontier for this theorem is the nonpositive Johnson gate, sparse-coverage
kernel, wild maps, or missing whole-petal anchors. The later coverage/kernel
tradeoff absorbs that kernel into growing partial-loss excess.
