# Result

The only received-interaction data in an arbitrary-rank first-core packet is

```text
rank 0: complete augmented orthogonality;
rank 1: one interactive row and t<=2r-1;
rank 2: slope-determined conic quotient.
```

All remaining row-space dimensions lie in
`(W+<b,q>|X)^perp`. The XR aggregate can therefore target
augmented-orthogonal lifts plus the already explicit rank-at-most-two
quotient, rather than independent interaction geometries at every rank.
