# Audit - L1 deep exact-shell Johnson closure

## Checked axes

1. One `m`-subset is selected per codeword only for the packing proof.
2. Pairwise support intersections are bounded by `k-1`.
3. The denominator is required positive before division.
4. The threshold uses `n+k-1`, matching the balanced-band boundary.
5. The arithmetic-mean/geometric-mean comparison uses `k-1`.
6. The deep tail is charged once through `L_>=m_J`, not once per level.
7. Exact shells are disjoint but only containment in the threshold list is
   needed.
8. The `n^2` bound is coarse and field-independent.
9. No claim is made in the balanced band.

## Route effect

The all-shell L1 reduction now has complete range coverage: a universal
Johnson payment above the band and one guarded balanced split-pencil problem
inside it.
