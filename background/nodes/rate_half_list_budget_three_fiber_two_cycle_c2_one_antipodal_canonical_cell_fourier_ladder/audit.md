# Audit

1. The cell elements `a` are the multiplier roots in
   `G_i(z)=product_(a in A_i)(1-az)`, not the polynomial zeros `a^(-1)`.
2. The annihilation range is strict: orthogonality through `w^s` gives
   `j<(s+1)H`; the next block can contain `w^(s+1)`.
3. Exact cell degree uses `deg Q=4r`; it is not inferred from a generic
   leading-coefficient assumption.
4. The support alternative includes `u_lambda=0`.  The Vandermonde argument
   cannot exclude an exactly negation-invariant weighted coloring.
5. The even-support improvement uses odd characteristic and the fixed-point
   free action `a->-a` on `mu_N`.
6. This is a canonical-span route.  It does not prove the primary `(P,Z)`
   circuit empty and does not prove `C2-PAR`.
7. No official-order enumeration or Modal run is needed for this theorem.
