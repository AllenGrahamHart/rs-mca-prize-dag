# Result

For each dyadic order, all rich product/quotient overlap is supported on the
prime divisors of one explicit odd cyclotomic derivative-ideal integer:

```text
p^X_18 divides e_n.
```

Hence `X_18=0` for every nonexceptional row prime, including primes inside
the previously open middle-field corridor. The universal C36' problem is now
reduced, at each order, to bounding the valuations at a finite exceptional
set. No bound for that set is claimed here.
