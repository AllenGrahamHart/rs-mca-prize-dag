# Audit

- Coprimality is inherited from the disjoint locator blocks `A_0,A_1`; it is
  not an extra primitivity hypothesis.
- The pendant degree of `beta` is proved using the exact degree-`d` locator
  `A_3`; possible leading cancellation in `A_1-hA_0` is not assumed away.
- Directness uses `d>=4`, so `deg alpha=d-2>=2`. This includes every official
  row and does not overclaim the small `d=3` boundary.
- The coefficient matrix in `(QPM4)` is the already-proved matrix `C`; no new
  coordinate normalization is hidden.
- A four-dimensional locator span is not an exclusion. The primitive
  four-moving-target factorization remains the arithmetic frontier.
