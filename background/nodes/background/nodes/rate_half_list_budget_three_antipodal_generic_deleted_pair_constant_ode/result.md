# Result

The surviving deleted-pair problem now has an original-coordinate form that
uses half-degree polynomials and no free generic direction `U_0`: for a fixed
deleted quadratic `D_0`, the constant ODE determines at most one monic
`U_0`, and a single terminal coefficient decides whether it is polynomial.

This also fixes an important saturation boundary. Every candidate has one
simple root of `U` at zero, and it has no other repeated, zero, or
deleted-divisor root. A contributor classifier should impose this exact
stratum rather than search generic official-degree `U,V` arrays.

The split quadratic field branch remains open because the ODE does not yet
exclude the simultaneous square-pencil factorization or determine `V_0`.
