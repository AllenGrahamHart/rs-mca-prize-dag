# Audit

1. The source word is nonzero.  Without this hypothesis the zero word has
   vacuous factorizations with every split point.
2. The split point is reconstructed from the unique parity decomposition of
   the source word.  No hidden factor `n` is charged for choosing it.
3. The count uses exact agreement supports, not arbitrary subsupports.
4. Interpolation uniqueness applies because both support sizes exceed
   `k-1`.
5. The owner covers only exact levels `k+1` and `k+3`.  Summing an unrestricted
   binomial tail would not fit the printed first-scale allowance.
6. First-match application after the earlier owners prevents overlap; the
   theorem does not assert the three source predicates are intrinsically
   disjoint.
7. This is a source-word owner, not a per-chart payment and not the raw
   one-boundary `QCLOSE` count refuted in the quotient-closure scope audit.
