# Audit

- `R_X=G_X/X_0`, not `G_X/B_X`; it removes only zero-trace rows.
- Squarefreeness of each quotient follows because it divides a squarefree
  fixed product and is coprime to the other factor.
- The exceptional slope is not included among clean partitions when
  `D_*=1`.
- No exceptional factor is cancelled from `V_a`: exceptional trace
  nonvanishing has already eliminated that allocation.
- `E_bad` counts exceptional incidences at all bad rows, including a possible
  zero-trace boundary row.
- A zero-trace row contributes zero to the clean-incidence sum, justifying the
  restriction from `B_X` to `X_1` in `(ATP7)`.
- Positivity of each active `a_x` follows from nonzero `H_x` and
  `deg H_x<a_x`; it is not inferred from an average.
- The partition families remain open.
