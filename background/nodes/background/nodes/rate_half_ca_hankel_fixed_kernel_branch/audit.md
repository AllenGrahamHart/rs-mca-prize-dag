# Audit

- The hypothesis is constancy of the whole generic right kernel, not merely a
  fixed root of a moving principal divisor.
- Equality over `F(Z)` implies separate endpoint annihilation because the
  pencil is linear in `Z`; there is no specialization assumption in this step.
- Column farness excludes only common **split** locators. The proof does not
  incorrectly exclude arbitrary common kernel vectors.
- A nonzero generic-rank minor controls every rank-drop slope and has degree at
  most `rho`; no point count over the split-locator set is used.
- The theorem covers fixed principal-divisor normal forms as a special case,
  including boundary generators, because such a normal form defines a fixed
  kernel subspace.
- The theorem deliberately leaves all genuinely moving generic kernels open.
  It imports no conditional SPI point-count theorem.
