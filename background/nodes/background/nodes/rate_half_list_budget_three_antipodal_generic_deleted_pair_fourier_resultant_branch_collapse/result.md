# Result

Two of the three maximal-row arithmetic branches are removed from the
complete deleted-pair sublocus. The prime-field branch is too large for the
split resultant bound, and Frobenius makes the nonsplit quadratic branch too
large for the stronger bound.

Only the split quadratic branch remains for the two-cell square-pencil
factorization. Its exclusion needs an argument beyond the primitive
resultant magnitude bound, since the official lower bound on `p` is below
`4N^2`.

That branch has no genuinely quadratic quotient-pencil coefficients: both
cells, both square-pencil parameters, all outer roots, and the Möbius map are
defined over `F_p`. The unresolved quadratic content lies only in the larger
evaluation domain.
