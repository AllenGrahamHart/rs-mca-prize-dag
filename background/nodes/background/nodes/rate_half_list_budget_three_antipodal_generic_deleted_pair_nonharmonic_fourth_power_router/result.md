# Result

The last dependence on a chosen outer-ratio root has disappeared. The three
branches now use only the source trace `chi`, the ODE-generated Euclidean data
`S,T`, a 38-step scalar trace gate, and the single branch-independent question
whether `T` is an exact fourth power.

CR-002 should request a compressed certifier for the three cleared identities
in `(NFR7)` followed by an exact fourth-power certificate. It should not ask
contributors to construct `q_out`, choose between reciprocal roots, or run a
separate square test for `S/(1+q_out)`.

Uniform rejection remains open. A large run is not authorized until a pilot
demonstrates a compressed representation of the official-degree remainder
and an explicit resource ceiling.
