# Audit

- Component row deficits `D_i` count missing distinct residual-domain roots;
  component column deficits `C_i` count missing distinct supported slopes.
- Component overlap `E` is retained. Since every nonexceptional supported
  fiber is squarefree with disjoint component roots, overlap can occur only at
  `lambda=0`.
- `sum D_i=O-E` can be zero when `O=E=1`; omission and overlap are not
  conflated.
- The factor-five formulas use exact `D_0=8e+7` and `T=4e+1`.
- Official congruence `e=3 mod 5` makes the dominant norm residual at least
  two even at maximum residual parameter degree.
- Norm exponents use distinct roots. Parameter tangency can only increase the
  residual factor and is included in `S_i`.
- Complementary factors are interpolated component by component; they are not
  claimed to be apolar generators.
- The remaining object is explicitly open.
