# Result

The nonlocal rank-two residue is now a low-degree rational-fiber problem.
The seven-coordinate profile is an exact quadratic pullback with three
paired fibers under one Mobius involution. The eight-coordinate profile is
either the same quadratic structure with four pairs or a cubic rational
fiber configuration.

All three cases have exact finite-field witnesses. The next rank-two work is
therefore an ownership theorem for local quadratic pullbacks and a separate
source test or count for the cubic eight-coordinate branch, not further
generic rank or overlap optimization.

No Modal resource is used and no XR critical status is promoted.
