# f3_h3_quotient_block_identity

- **status:** see `dag.json` (single source of truth)

For nonidentity `t`, put

```text
u(t)=1/(1-t),
v(t)=-t/(1-t).
```

Partition `F_p\{0,1}` by the coefficient coset `(u(t)H,v(t)H)`. If `B`
is the block containing `t`, then

```text
R(t)=|B|-1,
sum_B |B|(|B|-1)=(n-1)(n-2).
```
