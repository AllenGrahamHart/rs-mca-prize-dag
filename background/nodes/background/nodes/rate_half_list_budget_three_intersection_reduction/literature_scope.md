# Literature scope

Shangguan and Tamo introduce 4-wise intersection matrices in their proof that
generic Reed-Solomon evaluation vectors are optimal for list size three:

- https://arxiv.org/abs/1911.01502

Brakensiek, Gopi, and Makam relate the same list-decoding property to
higher-order MDS codes and prove it for generic Reed-Solomon codes:

- https://arxiv.org/abs/2206.05256

Neither generic theorem applies directly to the fixed multiplicative-coset
evaluation vector in the prize. The local theorem uses the same matrix
language only after deriving the exact finite rate-half predecessor types.
Importing a generic or high-probability statement would leave the critical
quantifier unproved.
