# Result

The exceptional `Z_W` allocation is impossible. A zero exceptional trace can
occur only when every domain trace is active, and then `E_Z` divides `B` and
the exceptional fiber is a squarefree degree-`r-1` divisor of `G_X`.

Every surviving active core now shares the full first complement
`QV_a+P_XW_a=P` and lies in one of the three exact systems `(ETA4)`. No Modal
computation was used.
