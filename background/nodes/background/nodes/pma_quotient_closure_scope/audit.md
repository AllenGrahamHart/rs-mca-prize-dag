# Audit - PMA quotient-closure scope

## Consumer-backward checks

1. **Objects.** Folding is stated for an actual received word and actual
   codeword, while low defect is stated for the full error support of an
   actual listed codeword. No auxiliary-list surjectivity is assumed.
2. **Folded receiver.** The receiver factorization is load-bearing. A folded
   polynomial alone need not have periodic agreement with an arbitrary word.
3. **Two support notions.** `QOWN_per` uses the agreement set. Boundary
   stripping uses its error-support complement. They coincide only through
   complementation at exact periodicity; the proof does not identify them at
   positive defect.
4. **Primitive guard.** Filtering preserves nonzero error amplitudes on the
   full core, so the periodic multisequence count has the required primitive
   guard.
5. **Route-cut direction.** `Raw_(2,1)>719Q_2` is a statement about the
   word-free candidate-universe upper route. It is not a lower bound for an
   actual fixed-syndrome class and is not a counterexample to PMA.
6. **Finite scope.** The `719` comparison is only for the official dyadic
   `sigma=1` grid. The boundary-stripping identity itself is general.
7. **Exhaustiveness.** With `QCLOSE` empty, every exact-stabilizer-one object
   either receives a separately proved source owner or enters B11.
   `J/A2/AR/GROW/RES` are already a proved partition on the remainder, so no
   class is dropped.
8. **Remaining debt.** The direct `GROW union RES` count and its composed
   allowance remain open. This node does not promote `pma_wide_residual`.

## Mutation guards

- dropping the folded-receiver hypothesis;
- claiming the one-defect raw route fits `719Q_2`;
