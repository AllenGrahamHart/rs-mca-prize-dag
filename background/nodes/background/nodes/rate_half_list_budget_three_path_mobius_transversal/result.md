# Result

The path-plus-singleton incidence type is no longer an unspecified coupled
pencil. It consists of two exact three-member constant pencils whose
remaining domain blocks cast injective Mobius parameters. At `d>=3` and
`d>=6`, respectively, neither pencil has a fourth fully split member. The
order-eight singular witness is the sharp `d=2` exception. Existence of the
large primitive three-fiber map remains open.
