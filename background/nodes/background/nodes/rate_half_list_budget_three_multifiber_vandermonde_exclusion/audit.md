# Audit

The proof uses only the highest `m` coefficient interval. It does not require
the quotient-value sets `C_i` to be disjoint, although completed agreement
blocks are disjoint in the consumer application.

The exceptional roots must be distinct for the Vandermonde step. They lie in
disjoint completed blocks in the cycle/path construction, so this hypothesis
is automatic there.

The application requires every relevant completed block to be a union of the
same number of fibers of one common map `X^m`. A construction using different
maps, partial fibers, or fiber sizes one or two is not silently included.

The theorem contradicts a locator relation already proved necessary by the
linear Grassmann atlas. It does not infer witness nonexistence from locator
independence in a chamber where no such constant relation is available.
