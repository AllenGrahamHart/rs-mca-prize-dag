# Audit

The edge weights are deficit weights in the centroid inequality, not quotient
multiplicities. Distance four contributes two units and distance six one unit
relative to the distance-eight baseline.

The weighted-degree conclusion uses seven selected vectors in one common odd
prime fiber. It does not hold for an arbitrary set of coefficient vectors
unless distance two is excluded.

A minimal incident set reaching weight four has at most four leaves and at
least two because each edge has weight one or two. No assertion is made that
all leaves have one distance class.

Adding ideal generators enlarges the ideal and decreases its norm. Therefore
the weighted-multistar candidate set is contained in, not larger than, the
two-generator ideal-star candidate set.

The fixed-root criterion is a screening theorem. It still requires a complete
principal candidate-prime list from an independent algebraic generator.
