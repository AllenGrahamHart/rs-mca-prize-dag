# Audit

The normal form counts slopes, not witness codewords. Multiple `p` values for
one slope are harmless in `(AP1)`; the official CA-only range has
`2r<d_min`, so the witness is in fact unique there.

The off-anchor set `T` excludes `E_0`. This is load-bearing: the column
distance is `s+t`, while the radius error word may cancel inside `E_0`.
Equation `(AP3)` has a plus sign because
`e_0+lambda(f_2-p)=0` implies `p=f_2+e_0/lambda`.
