# Audit

The conclusion is about a two-coordinate vector in the cyclotomic half-basis,
not an arbitrary polynomial presentation before reduction modulo
`Phi_n(X)=X^(n/2)+1`.

The nonzero condition is load-bearing. A same-coordinate cancellation has
norm zero, but shifted-product Sidonicity excludes it for distinct unordered
root pairs.

The packing improvement uses the finite-row fact that every pair of
representations in the selected rich fiber has collision norm divisible by
the same odd row prime. It is not a statement that distance two never occurs
between arbitrary characteristic-zero shifted products.
