# Result

The only explicit path witness on a power-of-two multiplicative domain is now
arithmetically isolated. Its fixed exponent pattern has a full minor whose
norm is `2^170 17^4`, so the rank defect cannot survive in any odd
characteristic other than `17`; that characteristic is impossible at the
prize-max order under the official field cap.

This removes the known finite witness as an official transport threat. The
actual `B*=3` frontier remains the classification or exclusion of all other
primitive path assignments together with the other twelve bounded chambers.
