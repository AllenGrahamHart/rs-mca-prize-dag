# Audit - reciprocal-quadratic PMA obstruction

1. This is an actual source construction. The codeword
   `f=L_(C\D)W` is written explicitly; no surjectivity of the general PMA
   auxiliary-list injection is assumed.
2. The petals are arbitrary disjoint pairs, as permitted by the maximal
   sunflower source. They are not claimed to be quotient fibers.
3. Distinct, nonzero labels follow from the index-two coset calculation, not
   from a genericity assumption.
4. Exact core defect uses `W(d)!=0`; exact background uses the separate
   averaging over `b`.
5. Diffuseness checks every petal: selected petals are protected by the
   matching bound, while an unselected petal already fails at its `X`
   endpoint.
6. Trivial stabilizer is proved from one antipodal asymmetry. In a cyclic
   two-group every nontrivial stabilizer contains `-1`.
7. The construction refutes only the stronger universal per-sunflower
   `n^6` payment route. The proved index-two near-core successor does pay it
   globally, so no `imgfib` counterexample is claimed.
8. The earlier general-pair census was source-admissible. The added antipodal
   rows are a useful house-chart slice, not a correction narrowing PMA's
   quantifier.
