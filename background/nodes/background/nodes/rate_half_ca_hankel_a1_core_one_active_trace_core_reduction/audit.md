# Audit

- `X_0` records identically zero polynomial traces, not sampled zeros.
- The impossibility of `J_x=0` uses the semantic consequence
  `Qhat_x|E_Z => epsilon_x=1`; the false `epsilon_x=0` escape is not retained.
- Zero exceptional traces use the already-proved `Z_W/Z_B` allocation. An
  active exceptional trace leaves `E_Z` in `E_a` and divides no biform.
- The three cancellations producing `(ACR4)` use disjoint squarefree factors.
- At a zero domain trace, equation `(1)` excludes every clean supported root.
  The exceptional root is still allowed and is charged separately.
- The strict bound excluding `z>=2` uses the official `e>=2`; the verifier
  checks it over bounded analogues.
- In the `epsilon_0=1` boundary, the remaining capacity is exactly one, so
  there is one further bad row rather than merely at most one.
- The active cores remain open.
