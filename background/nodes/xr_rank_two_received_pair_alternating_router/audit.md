# Audit

1. Pointwise agreement is used only to derive one parity equation for the
   selected trade row. The theorem does not infer all `h` block checks from
   that one equation.
2. The kernel word `w_i` disappears because `lambda_i` is dual to `GRS_a`
   on `X`, not because `w_i` is zero there.
3. The three-anchor branch has three constraints: the mixed coefficient is
   `<P,q>+<Q,b>`, not two separate zero conditions.
4. At least four distinct slopes are available, so the quadratic root
   argument is strict over every field.
5. The four-anchor conclusion uses coefficient rank four, not merely four
   active rows.
6. The `eta!=0` statement is only a perfect interaction pairing; it is not a
   tangent or quotient classification.
7. The converses certify trade-row parity only and retain all received-pair
   support-realization obligations.
8. No Modal or candidate-row computation is used.
