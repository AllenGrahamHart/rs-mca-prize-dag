# petal_squarefree_classification_ledger_soundness

- **status:** PROVED
- **closure:** proof

## Statement

If a ledger covers every squarefree realizable locator point in the
residue-line kernels by disjoint charged or uncharged class records, every
charged record cites an already paid family, every uncharged record has a
polynomial bound with exponent independent of `c`, and the number of
uncharged records is independent of `c`, then
`petal_squarefree_kernel_classification_payload` holds.

## Falsifier

A complete valid classification ledger while the squarefree-kernel
classification payload fails.
