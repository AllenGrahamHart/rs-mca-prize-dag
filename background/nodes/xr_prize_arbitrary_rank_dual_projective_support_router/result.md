# Result

All nonterminal trade ranks now share one record type:

```text
dual r-space L subset (W|X)^perp,
base-point-free Phi:X->P^(r-1),
t coefficient hyperplanes with r+2<=t<=2r+1,
interaction chamber of dimension 2r-t+1.
```

At maximal arity the packet is augmented-orthogonal. Repeated coefficient
directions route to basis/flat owners. The remaining higher-rank work is an
aggregate count of these first-match projective packets.
