# Audit

The compiler uses the full fixed side `X`, not only a subset of it. Replacing
`Lambda_X` by a partial locator changes both `K'` and the exact excess.

Deleting `Y` from the reduced domain is legitimate because every residual
`I` is disjoint from both oriented sides. The reduced dimension remains
`K-t`, and nonempty fibers guarantee `A'<=N'`.

Global genericity is proved after reduction, not assumed. Any reduced joint
explanation lifts through `a_X+Lambda_X q_0` and
`b_X+Lambda_X q_1` to an original `A`-support explanation.

The boundary payment is per fixed difference. Multiplying it by the number
of possible `(X,Y)` would destroy the target and is not claimed.
