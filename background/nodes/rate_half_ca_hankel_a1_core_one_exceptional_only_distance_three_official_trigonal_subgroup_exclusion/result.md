# Result

The official rank-three distance-three packet cannot carry the unique
trigonal Schur defect. A degree-three recovery map would put `2e` ordered
exceptional pairs on one subgroup coincidence curve, while explicit
bidegree-`(2,3)` bounds allow fewer than `2e`. The geometrically reducible
case would require an order-three Mobius deck map, also impossible on the
order-`2^41` subgroup. Therefore the official generic quadratic matrix has
full allowed rank `3e+1`; only its saturated branch remains.
