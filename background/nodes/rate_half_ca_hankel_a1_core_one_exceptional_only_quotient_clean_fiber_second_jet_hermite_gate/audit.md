# Audit

1. The derivation uses the global reversed first-complement identity, not an
   invalid division by `R_X` or `P_cl` at their common zero.
2. The second derivative is taken along the moving root curve; `(QSJ3)`
   separates total and partial derivatives.
3. Division in `(QSJ2)` is valid because the domain root is simple, the
   crossing is transverse, and the official field has odd characteristic.
4. `j_gamma` is unique because there are exactly `r` distinct values and its
   `Y`-degree is at most `r-1`.
5. The term involving the derivative of `tA_W+B_W` vanishes at a root of
   `P_cl`; only `P_cl'(gamma)(gamma A_W+B_W)` remains.
6. The theorem adds an independent compatibility gate. It does not infer
   rejection from the number of constraints alone.
