# Audit

1. `Beta` is only an interpolation class modulo squarefree `A`; its norm is
   well-defined and equals the product of the printed source weights.
2. `B_T` cancels pointwise in `(2)` and is a unit modulo `A`.
3. The derivative product contributes `(-1)^e Disc(A)`, so no discriminant
   or sign remains hidden in `(HNC1)`.
4. `Res(A,q_1)` is nonzero and appears squared. Its endpoint square class
   therefore cannot strengthen this gate.
5. The fence concerns direct packet evaluation. It does not forbid an
   independent endpoint-profile identity from forcing a contradiction.
