# Audit

The singular branch is not declared tangent-paid for `h>=3`. It supplies only
`k+2` common agreements, so its exact unresolved deficit is `h-2`.

The absence of a mixed branch uses distinct slopes and `t>=4`. One common
root of the two affine facet forms makes their quotient constant at every
other selected facet; this would repeat a slope unless both forms vanish
identically.

The regular relation space is complete only for active union `a+2`, where
every row is a facet word. Larger active unions can carry polynomial-pencil
relations not generated by these local face syzygies.

The node quotients a known local relation space; it does not assert that the
corresponding selected slopes have already been counted by the XR budget.
