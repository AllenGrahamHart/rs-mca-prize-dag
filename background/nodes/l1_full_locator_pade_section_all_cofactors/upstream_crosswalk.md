# Upstream crosswalk - all-cofactor Pade section

## List-interior object

For every exact-shell degree, including `e>=k`, the arbitrary-word list
interior is a split-divisor intersection with the `w` equations

```text
[T^(e+1..e+w)] Uhat/Lhat = 0.
```

This gives a literal candidate owner object for upstream `U_list-int`.  Below
`e=k` it is the prefix Pade graph cylinder; above the cap it is a continued-
reciprocal section of the full locator coefficient space.

## Scope against Q

The active upstream row-sharp Q atom fixes one prefix target.  The all-
cofactor section is not that atom and is not implied by image-normalized Q.
It should be presented as the list-interior completion problem, with quotient
and field-drop profiles removed first.

## Compute handoff

Contributor packets should emit the `w` reciprocal coefficients, recovered
cofactor, split locator, complement gcd, first owner, and exact section count.
For `e>=k`, a packet that records only the fully determined locator per fixed
cofactor discards the useful section structure and is not a sufficient
handoff.
