# Result

The antipodal DSP8 residue is an exact restricted multiplicative-energy
problem. For the Cayley image `B=C(H\{1,-1})`, richness at target `1-a^2`
is precisely

```text
#{(alpha,beta) in B^2:alpha beta=C(a)}>=23,
```

and `K_25^A` is the ordered signed-disjoint edge count in those fibers,
weighted by the original quotient-line multiplicity. This exposes a concrete
analytic route while retaining every load-bearing finite constant.
