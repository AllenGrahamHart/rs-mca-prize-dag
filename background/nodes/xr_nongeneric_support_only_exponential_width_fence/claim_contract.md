# Claim contract

- **claim:** at each clean-rate root, abstract equal-size support families
  obeying the exact pairwise `K-1` intersection cap can exceed `16n^3`;
- **scope:** only the six initial clean-rate parameter triples;
- **consumer:** `xr_tangent_support_mismatch_bridge` as a route fence;
- **dependencies:** none beyond the printed root arithmetic and the proved
  elementary Gilbert/entropy argument;
- **falsifier:** failure of `2A<=N`, `H/A<=1/16`, the binary-code support
  construction, or the budget comparison at one printed root;
- **nonclaim:** the abstract supports need not be realizable as RS joint
  explanations, so this does not falsify the bridge;
- **upstream mapping:** residual-ray compiler; pairwise support distance is
  not an adequate substitute for actual-ray or algebraic-realization control.
