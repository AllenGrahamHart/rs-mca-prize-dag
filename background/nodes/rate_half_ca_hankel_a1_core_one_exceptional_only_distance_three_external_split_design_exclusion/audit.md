# Audit

1. The pair-crossing gate is necessary for every exact external design, not
   merely for generic or sampled packets.
2. A kernel quartic for every omitted pair makes every crossing matrix
   deficient, so the all-deficient router is exhaustive.
3. The residual-discriminant theorem already includes bounded tails and all
   absorbed degree-four pullbacks; no unlisted support branch remains.
4. This composition does not address quotient distance greater than three
   or any other rate-half profile.
