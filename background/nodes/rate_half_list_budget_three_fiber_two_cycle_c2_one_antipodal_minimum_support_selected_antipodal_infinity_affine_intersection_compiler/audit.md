# Audit

1. The equal derivative-weight pair is the first quadratic in `(SAI2)`; the
   complementary pair has the smaller difference `2d`.
2. The two discriminants are `(6q)^2` and `(2q)^2`.  Reversing them changes
   both affine maps.
3. The fixed element satisfies `a^2=-2`, not `a^2=2`.
4. `tau` is a genuine subgroup scale fixed by the canonical top
   coefficients.  It is not removed from the source-product equation.
5. Eliminating `tau` gives `N/4`-torsion, not merely `N/2`-torsion.  The
   checked fixture attains exact order `N/4` at toy order 32.
6. The converse reconstructs only the infinity/top-coefficient packet, not
   the canonical recurrence, gaps, cells, or support divisor.
7. The `q -> -q` involution changes all of `a,d,u`; changing only `a` is not
   the same relabelling.
8. An explicit order-32 control passes the three affine subgroup tests, so
   the compiler is not an emptiness theorem.
