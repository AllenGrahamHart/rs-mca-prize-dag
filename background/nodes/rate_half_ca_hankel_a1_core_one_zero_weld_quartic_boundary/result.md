# Result

Every zero-weld profile except one is excluded. The sole survivor has no
residual component, one exceptional slope, and is a bidegree-one-quarter
factor of `F(U,V)-G_0(X)`, where both polynomials split and `G_0` omits exactly
three residual-domain factors.

The quartic separated boundary and the nonzero-weld branch remain open. No
Modal computation was used.
