# Result

The non-MDS endpoint is now an annihilator problem in the squarefree
exceptional algebra. Future work should show that the official normalized
split-incidence space contains no two independent elements with complementary
supports and zero product, rather than enumerate maximal minors. Canonical
gcd certificates split this into an excess-zero branch and an exact
factorization `A=D_uD_v` into complementary degree-`e` locators. Both gcds
can be computed directly from linear combinations of `q_1,...,q_e`; no
quotient-ring inversion is required. A half-rank deficiency `d` occurs on
both complementary halves and yields `d`-dimensional annihilator spaces,
hence `d^2` cross pairs rather than one isolated witness.
