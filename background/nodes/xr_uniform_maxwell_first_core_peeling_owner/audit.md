# Audit

1. The global density test `(PO1)` is only a sufficient trigger for a core;
   the selected core itself uses its actual union in `(PO2)`.
2. Passing to a residual deletes blocks only. It preserves every hereditary
   split-pencil hypothesis used by the parent theorem.
3. A declared total order is part of the compiler contract. No natural order
   on an abstract finite field is assumed.
4. A rank-two pivot is chosen outside the canonical coefficient basis, so it
   really has a unique fundamental-circuit owner.
5. A relation may be supported on a proper subset of its Maxwell core. The
   pivot is chosen from the active relation support and therefore belongs to
   the core.
6. Removing one pivot is enough to prevent the same pointed core certificate
   from recurring; no whole-core deletion is assumed.
7. The terminal bound uses strict failure of `(PO1)` and therefore contains
   `2N-2a-1`, not `2N-2a`.
8. The residual fits the allocation, but the produced certificate count is
   still the mathematical frontier.
9. No Modal or candidate-row computation is used.
