# Result

Any surviving dihedral external design is now a factorization of one explicit
orbit-level `e`th power into `3e` split degree-`r` factors. Unless the unique
printed identical-row orbit is active, every factor is squarefree. If it is
active, exactly `e` factors have that one double root and every other root is
simple. This all-or-nothing classification is the multiplicity interface for
a structured split-member argument.
