# Audit

- Checked that `E=0` is supported because `P=P_clE`; a boundary-only
  projective rank drop would not identify the finite exceptional factor.
- Used the exact squarefree degree `r-1` exceptional fiber to prove one
  omission, rather than assuming `O=1` merely from `D_*=1`.
- Used `O<=1` from the middle-adjugate theorem to make the omission slope
  unique and identify the two linear forms.
- Retained the scalar `c_H`; absorbing it into `q` could require a square root
  outside the base field.
- Primitivity proves only that some exceptional kernel coordinate is nonzero.
  The top coordinate vanishes by `q_r=E q_bar`, so the divided top row and
  column vanish while the full quotient remains rank one.
- No exact valuation claim assumes `gcd(E,q_bar)=1`.
